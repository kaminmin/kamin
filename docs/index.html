「仮眠」
とは何者か



略歴
2000年　 ゆらぎのある家庭に生まれる。
2006年　 何もわからないまま小学校に入学する。
2013年　 比較的賢い中学校に入学する。
　　　　 バンドに憧れ、エレキギターを手にする。
2015年　 哲学と出会う。
2019年　 高校を卒業する。
　　　　 慶應義塾大学に入学する。ダンスミュージック制作と哲学的随筆の発信をはじめる。
2021年夏- フランスの理工学系大学校に留学。現在に至る。　　



本質
僕自身の関心は一貫して 「やさしさ」というテーマの周辺をさまよっています。
僕の人生は、不条理や理不尽な事実を乗り越えることからはじまりました。人間関係においては多くの失敗(それはぼくの責任でも相手の責任でもないようなことです)をしました。そして、良くも悪くも、同質性の高い集団に収容されてばかりの自分と、それを良しとする周囲の人々に、嫌悪や抵抗の意志を向けることもありました。現に、僕はそのような集団から離れようと、ずっと努力してきました。
「やさしさ」とは、そのようなぼくの人生において得られた一つの処世術のようなものであり、また僕にとっての生き方の美学にもなっています。
「やさしさ」には常に対立する「つよさ」のようなものがあります(もしかしたら、対立していないかもしれません)。人間は、資本主義社会においてとにかくお金を稼ぎ、またけわしい実力主義社会において自らの価値を誇示しなければならない。そのような「つよさ」は、僕にとってある種の不条理であり、ある一面では人間の「大人らしさ」としてあらわれています。
「やさしさ」は、そのような人間の「つよさ」よりももっと和やかで、紳士的で、なにか生きるということの 本質に関わる意味を拾い上げてくれるものである、と僕は考えています。
こんな言葉があります。
強くなければ生きていけない。やさしくなければ生きる価値はない。
これはレイモンドチャンドラーの『ロンググッドバイ』に登場する一節で、しかもこれはある種の誤訳です。しかし、ぼくはこの言葉がとても好きで、色々な場所でこの言葉を引用しています。僕にとって「やさしさ」という概念は、生の意味の探究を表してもいるし、社会を生きて社会に価値を示すための意味をも表しています。生の意味を考えるということは、ひどく根源的であり、(もしかしたら)答えのない営みです。しかし、そこには価値がある。そして、もしかしたらそれほど根源的な営みでもないかもしれない。それぞれの人が納得できるまで考えればいいわけですから...。僕の精神の本質は、 「やさしさの探究」、そのような美的志向性にあります。
このような僕の哲学は、常に発信する必要があると感じています。だからこそ、このページにこうして言葉を書いているわけです。自らの「やさしさ」を「つよさ」とともに表現すること。そして、そのような自己が充実な生の意味にたどりつくためには、社会に「やさしさ」が表現されることも必要です。そのためには常に対外的であり、自己と社会の対話が必要であると感じています。このページはそのような発信と受信のために存在しています。

少しだけ 余談を付しておきます。
哲学史上、「やさしさ」は多くの視点から語られる機会がありました。ぼくは多くの哲学者の思想を知り、各所からその影響をうけています。特に、僕の哲学スタイルは20世紀に生まれた現象学と呼ばれる伝統に多く依拠しています。
現象学の創始者フッサールは、独我論的前提における世界の見え方を三つに区分しています。ほとんどのドクサ(臆見)が存在せず、我々の不可疑性の根拠、本質直観と知覚直観が垣間見れる〈生活世界〉、ドクサが多く含まれるものの、原理上経験可能ではあるようなものとして我々が「知っている」〈伝聞・情報の世界〉、そして、誰にとっても経験は不可能で、ドクサからのみで考えるしかないような〈フィクションの世界〉です。フッサールはこのうちの〈生活世界〉を主題にして、世界を確信する条件を問いただしました。現象学的還元とはここから発想されているものであり、フッサールの偉大な思考がみてとれます。
ぼくは現象学の方法を興味深く観察しています。そして、それは僕の「やさしさ」というテーマにも直結するものがある。
現代を振り返ってみると、世界は〈伝聞・情報の世界〉における可疑性にとらわれてしまっています。つまり、何を信じていいのか・何を疑わなくて良いのかということがわからなくなっているということであり、社会はそのような問題に何かしらの答えを見つけ出す必要があると言えるでしょう。僕の「やさしさ」という思想は、「可疑性を孕みつづける情報という概念をどのようにやさしく扱うべきか」という問いに接続します。現象学という物事の本質(あるいは確信の条件)を明らかにする手法に依拠することもあるでしょう。いづれにせよ、ぼくはそのような具体的問題意識を抱えています。

加えて もう一つ。
僕は音楽とともに多くの時間を過ごしてきました。しかし、それは決して高尚なものではなく、極めてポップな(そしてニッチな)ものです。中学高校時代はロックバンドをたくさん組み、多くの学外の人間と交流する機会がありました。それは僕の後の哲学的思考にも影響を与えたわけですが、それだけでなく、僕自身の音楽的な側面を切り開くことにもなりました。
大学入学と同時に DTM(デスクトップミュージック)、いわゆる作曲活動をするようになり、これまでに(ダンスミュージックに分類分けされるような)音楽をくりかえし聴いて、そしてくりかえし作りました。これまでにHardcore,FutureBass,DnB,UKG,その他の多くのBassMusicから影響を受けています。
ぼくにとって、音楽とは一つの表現方法です。ある種の社会との対話でもあり、それでいて芸術的な言語としても機能している。需要のある音楽とは何か、であるとか、今流行りの音楽は何であるか、とかいったことが常に念頭に置かれていて、そのような社会からの要求に対して、僕なりの芸術的返答を加える。それこそが楽曲制作の愉悦であり、一番たのしいことでもあります。
音楽というのはある意味で表現の一つに過ぎません。とはいえ、僕はどうしようもなくこの音楽という言語が好きです。ダンスミュージックは踊るために存在していますが、そのような音楽を体で浴びるという体験は、あまり「まじめ」な理由で愉しいというわけでもなく、とにかく「ふまじめ」に、肉体的に、感情的に、楽しいものなのです。時に音楽の喜びは芸術的に表現されることもあります。けれど、音楽というのは常に大衆的でもあります。それは常に身体的な喜びとも繋がっているわけです。そして、僕はこの音楽というの言語が大好きなのです。

僕はこの 言語という側面で、ほかにも別のものを獲得しようとしています。
第一に、中学高校から学んでいる英語です。良くも悪くも、僕はすごく英語ができるわけでもなければ、ぜんぜん英語ができないわけでもありません。日常会話や少し込み入った自分の趣味の話などであれば、たやすく英語で話せます。しかし、それを用いて日本語のように話せるわけでもないし、語彙は決して多いわけではありません。なので、これからも勉強しなければならない物事のひとつです。
第二に、大学に入ってから学びはじめたフランス語です。前述しているように、ぼくは哲学がとても好きです。そして、そのためにフランス語を学ぶことは、妥当な決断であったと感じています。2021年夏より、僕はフランスのとある大学に留学することとなりました。これまでの学習を活かす時が来た、ということです。いずれにせよ、僕は一年生の頃からフランス語学習をはじめて、いまでも続けていて、先日フランス語検定準2級を取得しました。
第三に、学科で学んでいるプログラミング言語です。学科では複数のプログラミング言語を学ぶ機会が設けられています。僕は特に芸術的表現にも近いメディア的なコンピュータ技術に興味があります。その中でもコンピュータグラフィックスに関心があります。



以上のようなメッセージが、僕の人生と思考を説明する上で役立つであろう情報になります。
僕のことが気になった方は是非とも、連絡してみてください。お話ししましょう。

クラブで音楽を聴いていると、よく気の利いた人がお酒を奢ってくれることがあります。そんな時、大抵おごってくれる人はとてもいい気分でクラブの空間を楽しんでいる。僕はそんなクラブでのひと時がとてもやさしい世界だと思います。そんな、とてもやさしい世界はどこに生まれて、どのようにして生まれるのか。そういうことが探しだせればいい。そんなふうに思っています。
